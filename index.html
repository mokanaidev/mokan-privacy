Privacy Policy for Mokan.ai

Last updated: April 22, 2026

1. Scope

This Privacy Policy describes how Mokan.ai (the “Extension”) processes data when the user creates UI mockups, generates requirement text, and optionally connects third-party services such as Trello and AI model providers.

2. Data Collection

Mokan.ai does not sell personal data. Data is processed only to provide extension features requested by the user.

Depending on feature usage, the Extension may process:

  • Active page context (URL, page title, and DOM-related metadata needed for mockup features).
  • User-created mockup content (widgets, layout metadata, screenshots, and related JSON model data).
  • Configuration data stored by the user (for example Trello token, selected AI model, and optional API key).

3. Use of Information

Processed data is used exclusively to:

  • Create, edit, and save mockups and requirement artifacts.
  • Generate or refine user stories and acceptance criteria through the AI model selected by the user.
  • Export or sync selected project artifacts to Trello when explicitly requested by the user.

4. Data Storage and Retention

The Extension stores configuration and workflow data in browser extension storage (chrome.storage) to preserve settings and user progress.

Users can clear local extension data by removing the extension or resetting extension options. Data sent to third-party services (e.g., Trello or AI providers) is then subject to those services’ own retention policies.

5. Third-Party Services and Data Transfers

Mokan.ai communicates with third-party services only when the user enables and uses those features:

  • Trello (Atlassian): when saving/opening mockups, selected data (e.g., JSON model, screenshots, card metadata) is sent to Trello APIs using user-provided credentials.
  • AI Model Provider selected by the user (e.g., Google Gemini): prompts and relevant context data are sent to the selected model endpoint to generate requirement text.

Please review third-party privacy policies:
- Atlassian/Trello: https://www.atlassian.com/legal/privacy-policy
- Google Privacy Policy: https://policies.google.com/privacy

6. API Usage, Model Selection, and Potential Costs

Mokan.ai allows the user to choose an AI model/API provider. Requests are sent to the selected model according to the user configuration.

Important: depending on the selected API/model and the user’s provider account plan, usage may incur charges. Any billing relationship exists directly between the user and the provider (for example Google/AI Studio or related cloud billing account).

7. Security

Mokan.ai is designed to limit processing to what is required for requested features. However, no internet transmission or third-party platform can be guaranteed as 100% secure. Users should avoid sending sensitive information unless necessary for their workflow.

8. Children’s Privacy

The Extension is not directed to children under 13, and Mokan.ai does not knowingly collect personal data from children.

9. Changes to This Policy

This Privacy Policy may be updated from time to time. Any update will be published at this URL with a revised “Last updated” date.

10. Contact Information

For privacy-related questions, contact: privacy@mokan.ai
